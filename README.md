## 🚀 사용 예시 (Usage)

`ip_list.txt` 파일에 검사할 IP 주소들을 한 줄씩 입력한 뒤 스크립트를 실행합니다. 스크립트는 AWS의 최신 IP 대역 정보를 실시간으로 가져와 CloudFront 대역 여부를 판별하고, 그 결과를 터미널에 출력함과 동시에 `.csv` 파일로 자동 저장합니다.
